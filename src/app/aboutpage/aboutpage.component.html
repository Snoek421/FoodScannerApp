About the App

This app was completely written by Casey Snoek.

This app was developed as a final project for:
PROG2430 - Programming Mobile Application

I wanted to make this app so I could use it for myself and continue developing it in the future.
There are not many apps that are made to scan a list of ingredients and check for anything containing gluten, so I decided I should make one myself. I wanted to have the ability to extract text from an image of an ingredients label but could not get that working on the available timeline, so I hope to add that feature eventually.

For a new product...
  1. Select a profile or create one with your name and your dietary restrictions.
  2. Click the "Add New Product" button or "Add Product" in the navigation menu.
  3. Enter the product name and the ingredients list from the product's label.
  4. Take a picture of the ingredients label if desired
  5. Click the "Scan product" button.
  6. Read the results page to check if any ingredients are found that trigger your chosen dietary restriction(s).
For a previously added product...
  1. Select a profile or create one with your name and your dietary restrictions (or modify your profile to change your dietary restrictions).
  2. Select a product from the list of previously added products.
  3. Click the "Re-Scan Product" button. It will bring you back to the list of products.
  4. Select the same product again to view the new scan results.
